# Pages: About, Services & Contact

Documentation of the **About**, **Services**, and **Contact** experiences of Digital Moon Agency. In the current build these are **in-page anchor sections** of the single-route `HomePage` (`apps/web/src/pages/HomePage.jsx`), reachable from the header nav (`#about`, `#services`, `#contact`) with `scroll-behavior: smooth`. This file captures their exact design, animation, layout, component arrangement, styling, color, typography, spacing, and interaction specs.

> Note on About: the header exposes an `#about` link, but there is currently
> **no dedicated About section** in the markup. The anchor resolves to the
> nearest matching element if one is added later. This document specifies the
> intended About treatment (matching the established system) so it can be
> implemented consistently, alongside the shipped Services and Contact sections.

## Shared foundations (all three sections)

### Color tokens (from `apps/web/src/index.css`)
| Token | Value | Role |
| --- | --- | --- |
| `--background` | `5 5 5` (near-black) | Page canvas |
| `--foreground` | `255 255 255` | Primary text |
| `--card` | `15 15 15` | Card / panel surfaces |
| `--muted` | `26 26 26` | Muted surfaces |
| `--muted-foreground` | `163 163 163` | Secondary text |
| `--accent` / `--primary` | `hsl(114 96% 76%)` ≈ `#95FC89` | Neon-green accent |
| `--accent-foreground` | `0 0 0` | Text on accent |
| `--border` / `--input` | `42 42 42` | Hairline borders |
| `--ring` | `hsl(114 96% 76%)` | Focus ring |

### Typography & Spacing
*   **Section Padding:** All major sections share a vertical rhythm of `py-24` (6rem/96px) and horizontal padding of `px-6` on mobile, scaling up with a `max-w-7xl mx-auto` container for desktop.
*   **Headings:** Section titles utilize `text-3xl md:text-5xl font-bold tracking-tight mb-12`.
*   **Body Text:** Standard body text is `text-lg text-muted-foreground leading-relaxed`.

## 1. About Section (Intended Blueprint)

### Layout Structure
*   **Grid:** 2-column asymmetric layout (`grid grid-cols-1 lg:grid-cols-2 gap-12`).
*   **Left Column (Visuals):** A geometric image placeholder or branded graphic using an aspect ratio of `aspect-square` with `rounded-2xl bg-muted border border-border/50`.
*   **Right Column (Copy):** Stacked flex container (`flex flex-col justify-center space-y-6`). 

### Animation & Interactions
*   **Scroll Reveal:** Elements should fade in and slide up slightly (`opacity-0 translate-y-8`) when intersecting the viewport, utilizing an Intersection Observer with a `duration-700 ease-out` transition.

## 2. Services Section

### Layout Structure
*   **Container:** `max-w-7xl mx-auto px-6 py-24`.
*   **Grid:** Responsive 3-column matrix (`grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6`).
*   **Component Structure:** Maps over data array to render `<ServiceCard />` components.

### `<ServiceCard />` Design Specs
*   **Base Styling:** `bg-card border border-border/60 p-8 rounded-xl relative overflow-hidden transition-all duration-300 group`.
*   **Typography:** 
    *   Title: `text-xl font-semibold mb-3 text-foreground`.
    *   Description: `text-sm text-muted-foreground leading-relaxed`.
*   **Icon Treatment:** Uses an inner wrapper `p-3 bg-muted rounded-lg w-fit mb-6 text-primary`.

### Animation & Interactions
*   **Hover State (Container):** On mouse hover (`group-hover`), the card elevates vertically (`-translate-y-2`) and the border illuminates (`border-primary/50`).
*   **Hover State (Icon):** The nested icon background scales up subtly (`group-hover:scale-110 transition-transform duration-300`).

## 3. Contact Section

### Layout Structure
*   **Container:** `max-w-7xl mx-auto px-6 py-24`.
*   **Grid:** 12-column asymmetric split (`grid grid-cols-1 lg:grid-cols-12 gap-12`).
    *   **Left Panel (Info):** `lg:col-span-5` - Contains headline, subtext, direct contact info, and `<SocialIcon />` group.
    *   **Right Panel (Form):** `lg:col-span-7` - Houses the interactive submission form.

### Form Design Specs
*   **Wrapper:** Uses a glassmorphic elevated panel (`bg-card/40 border border-border/50 p-8 md:p-10 rounded-2xl backdrop-blur-sm`).
*   **Inputs & Textareas:**
    *   Background: `bg-background`.
    *   Border: `border-border/80`.
    *   Height/Padding: Inputs are `h-12 px-4`. Textareas are `min-h-[120px] p-4`.
    *   Radius: `rounded-lg`.
*   **Submit Button:** Solid primary accent block (`w-full h-12 bg-primary text-accent-foreground font-semibold rounded-lg hover:opacity-90 transition-opacity`).

### Animation & Interactions
*   **Focus States:** All input fields utilize a high-visibility accessibility ring upon interaction (`focus-visible:ring-2 focus-visible:ring-primary focus-visible:outline-none focus-visible:border-transparent`).
*   **Social Icons:** `<SocialIcon />` components use a subtle background fill and color shift on hover (`hover:bg-primary/10 hover:text-primary transition-colors`).
